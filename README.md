# sqquid-excercises
## Back-End
1. **Set up Node version 13.x**  
2. Run:
```bash
  cd ./backend
  npm i
  npm test
```

## Front-End
1. **Set up Node version 13.x**  
2. Run:
```bash
  cd ./frontend/todo-list
  npm i
  npm start
```
